1- Récupérer le projet ici : 
	https://github.com/dadarubik/angular-blog
2- Se placer dans le répertoire du projet
3- Exécuter la commande : 
	npm install
4- Puis
	ng serve
	
Pour info, dans le menu, j'ai ajouté la possibilité d'ajouter 3 posts automatiquement pour faire des tests.

Enfin l'application intègre un backend Firebase.
